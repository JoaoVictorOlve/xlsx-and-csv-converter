# RPA Challenge

Desafio RPA realizado com Selenium, Playwright e Pyautogui.
___

## Para acessar o projeto 
* Clone o reposit√≥rio no seu ambiente local com `git clone`.
* Acesse a pasta raiz do projeto.
___
## Para executar o desafio em Pyautogui 
* `cd Pyautogui`.
* `py -m venv .venv`
* `.venv/Scripts/activate`
* `py -m pip install -r requirements.txt`
* `py -m main`
___
## Para executar o desafio em Selenium 
* `cd Selenium`.
* `py -m venv .venv`
* `.venv/Scripts/activate`
* `py -m pip install -r requirements.txt`
* `py -m main`
___
## Para executar o desafio em Playwright 
* `cd Playwright`.
* `py -m venv .venv`
* `.venv/Scripts/activate`
* `py -m pip install -r requirements.txt`
* `py -m playwright install`
* `py -m main`
